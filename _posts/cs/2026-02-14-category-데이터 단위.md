# 컴퓨터 데이터 단위

비트(Bit)의 기본 개념부터 기가바이트(GB), 테라바이트(TB)까지의 확장 원리를 정리한 내용이다.

---

# 1. 데이터 단위 한눈에 보기

컴퓨터는 2진수 기반으로 동작하므로 데이터 단위는 기본적으로 1024배(2¹⁰)씩 증가한다. 단, Bit에서 Byte로 넘어갈 때만 8배가 된다.

| 단위 | 약어 | 관계식 | 예시 |
| --- | --- | --- | --- |
| Bit | b | 0 또는 1 | 전등 스위치 한 번 켜고 끄는 정보량 |
| Byte | B | 1 B = 8 b | 영문자 1글자 |
| Kilobyte | KB | 1 KB = 1,024 B | 짧은 텍스트 문서 |
| Megabyte | MB | 1 MB = 1,024 KB | 음악 파일, 사진 파일 |
| Gigabyte | GB | 1 GB = 1,024 MB | 영화, 모바일 게임 |
| Terabyte | TB | 1 TB = 1,024 GB | SSD, NAS 저장장치 |

---

# 2. 데이터 단위 확장 원리

## ① Bit → Byte

비트(Bit)는 컴퓨터가 표현할 수 있는 가장 작은 데이터 단위이다.

1비트는 0 또는 1의 두 가지 상태만 표현할 수 있으므로 문자나 숫자를 저장하기에는 정보량이 부족하다.

8개의 비트를 묶으면 256개의 값을 표현할 수 있으며, 이를 1바이트(Byte)라고 정의한다.

$2^8 = 256$

따라서 컴퓨터는 문자를 저장할 때 기본적으로 바이트 단위를 사용한다.

---

## ② Byte → KB

컴퓨터는 2진수 체계를 사용하므로 1,024(2¹⁰)를 하나의 묶음 단위로 사용한다.

1KB = 1024Bytes

즉, 1바이트 크기의 데이터 1,024개가 모이면 1KB가 된다.

---

## ③ KB → MB

1,024KB가 모이면 1MB가 된다.

1MB = 1024KB

MB는 텍스트뿐만 아니라 이미지, 음악 등의 멀티미디어 데이터를 표현하는 데 사용되는 단위이다.

플로피디스크 한 장의 용량은 약 1.44MB였다.

---

## ④ MB → GB

1,024MB가 모이면 1GB가 된다.

1GB = 1024MB

GB는 현재 가장 널리 사용되는 저장 용량 단위이다.

스마트폰 애플리케이션, 운영체제, 고화질 동영상, 게임 등의 용량을 표시할 때 주로 사용된다.

---

## ⑤ GB → TB

1,024GB가 모이면 1TB가 된다.

1TB = 1024GB

TB는 대용량 저장장치와 서버 시스템에서 주로 사용되는 단위이다.

최근에는 개인용 SSD도 1TB 이상이 일반적으로 사용되고 있다.

---

# 3. 데이터 단위 정리

| 단위 | 바이트 기준 |
| --- | --- |
| 1 Byte | 8 Bit |
| 1 KB | 1,024 B |
| 1 MB | 1,024 KB |
| 1 GB | 1,024 MB |
| 1 TB | 1,024 GB |
| 1 PB | 1,024 TB |
| 1 EB | 1,024 PB |

암기 순서는 다음과 같다.

> Bit → Byte → KB → MB → GB → TB → PB → EB
>

# 4. 관련 상식

## 저장장치 용량과 실제 표시 용량이 다른 이유

500GB SSD를 구매했는데 운영체제에서는 약 465GB로 표시되는 경우가 있다.

이는 저장장치의 이상이 아니라 계산 기준의 차이 때문이다.

### 제조사 기준

제조사는 10진수 기준을 사용한다.

1GB = 1000MB

따라서 500GB는 다음과 같이 계산된다.

500,000,000,000Bytes

### 운영체제 기준

운영체제는 2진수 기준을 사용한다.

1GB = 1024MB

따라서 실제 표시 용량은 다음과 같이 계산된다.

500,000,000,000 \div 1024^3 \approx 465.66

결과적으로 약 465GB로 표시된다.

즉, 저장장치가 불량인 것이 아니라 계산 기준이 다르기 때문에 발생하는 현상이다.

---

## 대문자 B와 소문자 b의 차이

| 표기 | 의미 | 사용 예시 |
| --- | --- | --- |
| B | Byte | 512GB SSD |
| b | Bit | 100Mbps 인터넷 |

저장장치 용량은 일반적으로 Byte(B) 단위를 사용한다.

반면 네트워크 전송 속도는 Bit(b) 단위를 사용한다.

예를 들어 100Mbps 인터넷은 초당 100MB를 전송한다는 의미가 아니다.

1Byte는 8Bit이므로 실제 전송 속도는 다음과 같다.

$100Mbps \div 8 = 12.5MB/s$

즉, 100Mbps 인터넷의 실제 다운로드 속도는 약 12.5MB/s이다.

---

# 정리

- Bit는 컴퓨터가 표현할 수 있는 최소 단위이다.
- 1Byte는 8Bit로 구성된다.
- Byte 이후의 데이터 단위는 1024배씩 증가한다.
- 데이터 단위의 순서는 Bit → Byte → KB → MB → GB → TB → PB → EB이다.
- 저장장치 용량은 Byte(B) 단위를 사용한다.
- 네트워크 속도는 Bit(b) 단위를 사용한다.
- SSD 용량이 실제보다 작게 보이는 이유는 10진수와 2진수 계산 기준의 차이 때문이다.
